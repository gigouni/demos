stay tuned?

## Deployment

_On remove server using SSH_

* Connect to your remote server
* Clone remotely the project
* Go to the genogram.html page
* Enjoy

_Without SSH but FTP_

* Clone locally the project
* Connect to your remote server using FTP
* Copy the files
* Go to the genogram.html page
* Enjoy
